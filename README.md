## About The Project

This is a professional portfolio website for **Teodorina Lungu**, a Senior Technician in DevOps and Data Strategy. The site showcases a "Performance Journey" connecting behavioral psychology with technical infrastructure.

It features a high-performance, Apple-inspired aesthetic with a dark/light mode, vertical timeline animations, and live project previews.

## Technologies Used

This project is built with a modern, performance-first stack:

- **HTML5 & CSS3** (Semantic structure)
- **Tailwind CSS** (Utility-first styling for the "Lovable" aesthetic)
- **JavaScript (ES6+)** (Interactive elements and DOM manipulation)
- **FontAwesome** (Iconography)
- **Google Fonts** (Inter & Playfair Display typography)

## How to Edit This Project

There are several ways to edit this application.

### 1. Edit directly in GitHub
1. Navigate to the file you want to change (e.g., `index.html`).
2. Click the **Edit** button (pencil icon) at the top right.
3. Make your changes and click **Commit changes**.

### 2. Work Locally (Recommended for DevOps practice)
If you want to work locally using your own IDE (like VS Code), follow these steps:
